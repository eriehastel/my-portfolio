# My Portfolio

This is my personal portfolio website, built with **React.js** and styled using **Tailwind CSS**. It showcases my skills, projects, and contact information.

## Live Demo
🔗 [Visit My Portfolio](https://eric-mugendi.netlify.app/)

## Features
- **Responsive Design**: Works on all devices.
- **Dynamic Animations**: Includes typewriter effects and smooth transitions.
- **Projects Showcase**: Displays my best projects.
- **Contact Information**: Direct links to my email, LinkedIn, and GitHub.
